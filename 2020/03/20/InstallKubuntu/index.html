教大家如何使用VMware安装Kubuntu系统

发表于 2020-03-20 更新于 2020-03-22 分类于 Linux

获取Kbuntu.iso

如果你已经有了Kbuntu.iso或者其他系统，那就不用下载了，直接跳到打开VMware

点击下载Kbuntu
您正下载的Kbuntu是
即64位的18.04 LTS版本的Kubuntu

  1. 为什么我用浏览器下载很慢？
    亲，这边建议您用迅雷下载！迅雷给你打了多少钱
  2. 为什么不下载官网的Kubuntu？
    因为官网的用的是国外的服务器，到时候安装Kubuntu系统时就会很慢慢~！
    您正下载的是南京大学开源站Kubuntu镜像源，到时候安装就不至于那么慢了！
    点击了解镜像网站
  3. 本教程是只针对Kubuntu的安装吗？
    并不是，你可以直接跳到打开VMware来进行别的系统的安装！
  4. 那有没有其他系统的下载链接？
    有的！点击查看其他版本的Kubuntu，点击查看其他系统
    这些也是南京大学开源站的镜像源。
    当然还有其他镜像源网站：网易开源镜像站、阿里云、更多镜像网站

打开VMware

本文的VMware是指Workstation Player！
如果你已经安装了VMware，那就跳到下一步

点击下载VMware
这里也建议使用迅雷下载，可能更快！
下载完后就安装，如果C盘位置不够就换个位置够的磁盘安装。
安装过程中那些选项都可以不用改。
安装完后，到如下图界面。

点击许可证，复制下面的序列码

然后粘贴到框框中，点击输入，就算安装并激活了！

接着打开VMware吧！

注意事项

  1. 如何获取其他版本的VMware？
    点击访问VMware官网下载列表
    然后找到VMware Workstation Pro点击下载产品。
    不过你得先注册一个账号才能下载
  2. 为什么序列号激活失败？
    可能官网限制了序列号的多次使用吧！
    要么百度换个序列号或者找破解方法试试看，
    要么去官网下载VMware Workstation Player使用个人版。
  3. 为什么不推荐使用VMware Workstation Player？
    因为这是免费版，也就意味着功能的削减！
  4. 我错过了许可界面怎么办？
    不用担心，在软件里面也可以激活
    点菜单栏的帮助，然后点输入许可证密钥就可以去激活了。
  5. 我如何查看是否激活成功？
    点菜单栏的帮助，然后点关于VMware Workstation，就可以看到激活信息啦！
